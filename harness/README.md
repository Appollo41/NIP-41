# NIP-41 Key-Rotation Confirmation Harness

Executes the NIP-41 draft's cryptographic construction and proves — against
independent reference code — whether it is correct. See
`CONFIRMATION-REPORT.md` (generated by `python report.py`) for results.

## Setup

    python -m venv .venv && source .venv/bin/activate
    pip install -e ".[dev]"

## Run

    pytest -v                 # full confirmation suite
    python cli.py selfcheck   # quick build + verify of one chain
